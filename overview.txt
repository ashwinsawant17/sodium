## 🧱 Architecture Overview

### 1. Core Layers

```
[ Applications ]  -> server/, client/
        ↓
[ Protocol Layer ] -> message format, serialization
        ↓
[ Networking ]     -> sockets, connections, event loop
        ↓
[ Utilities ]      -> logging, threading, buffers
```

---

## 🔌 Networking Design

### tcp_server.hpp

Responsibilities:

* Accept new connections
* Own event loop
* Dispatch connections to worker threads

### tcp_connection.hpp

Responsibilities:

* Manage a single client
* Read/write buffers
* Parse messages

### event_loop.hpp

Options:

* `select()` (simple)
* `poll()`
* `epoll` (Linux, recommended for performance)

---

## 🧵 Threading Model

Recommended: **thread pool + non-blocking I/O**

* 1 acceptor thread
* N worker threads
* Each worker handles multiple connections

Alternative (simpler):

* Thread-per-connection (not scalable, but easy)

---

## 📦 Protocol Layer

Define a simple binary or JSON protocol:

Example:

```
[ length (uint32) ][ type (uint16) ][ payload ... ]
```

Files:

* `message.hpp` → struct definitions
* `serializer.hpp` → encode
* `parser.hpp` → decode

## ⚙️ Root CMakeLists.txt

```cmake
cmake_minimum_required(VERSION 3.20)
project(TCPProject LANGUAGES CXX)

set(CMAKE_CXX_STANDARD 20)
set(CMAKE_CXX_STANDARD_REQUIRED ON)

add_subdirectory(src)
add_subdirectory(apps/server)
add_subdirectory(apps/client)
add_subdirectory(tests)
```

---

## ⚙️ src/CMakeLists.txt

```cmake
add_library(core
    common/logging.cpp
    common/error.cpp
    net/socket.cpp
    net/tcp_server.cpp
    net/tcp_connection.cpp
    net/event_loop.cpp
    protocol/message.cpp
    protocol/serializer.cpp
    protocol/parser.cpp
    util/buffer.cpp
    util/thread_pool.cpp
    util/config.cpp
)

target_include_directories(core PUBLIC ${PROJECT_SOURCE_DIR}/include)
```

---

## ⚙️ apps/server/CMakeLists.txt

```cmake
add_executable(server main.cpp)

target_link_libraries(server PRIVATE core)
```

---

## ⚙️ apps/client/CMakeLists.txt

```cmake
add_executable(client main.cpp)

target_link_libraries(client PRIVATE core)
```

---

## 🧠 Example server main.cpp

```cpp
#include "net/tcp_server.hpp"

int main() {
    TcpServer server(8080);
    server.start();
}
```
